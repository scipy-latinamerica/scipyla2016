# Brochure for SciPyLA 2016

## Directory structure

-   `examples`

    Contains examples of others brochures.

-   `images`

    Contains images used on the brochures.

## Compilation dependencies

-   LaTeX

## Compilation steps

### GNU/Linux and (Mac) OS X

~~~
$ latexmk -pdf template.tex
~~~

### Windows

~~~
$ pdflatex.exe template.tex
$ pdflatex.exe template.tex
$ pdflatex.exe template.tex
~~~
